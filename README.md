# codmin

=> **co**mic + a**dmin**

個人的コミック管理アプリケーション

---

##### ●実装環境
ruby: 2.3.1  
rails: 5.1.4 

---

##### ●使い方
1. TOPページにアクセスします  
    path: / 
2. ユーザー登録します  
    path: /users/new 
3. 登録したユーザーでログインします  
    path: /login 
4. コンテンツ検索&登録ページを開きます  
    path: /contents/new 
5. DMMのコミックコンテンツを検索します  
    path: /contents/new 
6. 登録したいコンテンツを登録します  
    path: /contents/new 
7. 詳細ページで登録した内容を確認します  
    path: /contents/:id 
8. タグや作者の一覧ページを確認します  
    path: /contents/:type/:name 
9. Happy!
